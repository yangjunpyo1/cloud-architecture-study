# Cloud Architecture Study

NHN Cloud 레퍼런스 아키텍처를 AWS 서비스로 변환하여 Terraform으로 구현한 실습 기록입니다.

## 사용 기술

![Terraform](https://img.shields.io/badge/Terraform-7B42BC?style=flat&logo=terraform&logoColor=white)

## 필수 아키텍처

| # | 아키텍처 |
|---|---|
| 01 | 3-Tier 웹 애플리케이션 아키텍처 |
| 02 | Auto Scale을 통한 효율적 리소스 관리 및 서비스 안정성 확보 |
| 03 | 고가용성 구성 |
| 04 | 가용성 영역을 활용한 비즈니스 연속성 보장 |
| 05 | Network Firewall 기본 구성 |
| 06 | NKS, NCS 기반의 CI/CD 파이프라인 구성 |
